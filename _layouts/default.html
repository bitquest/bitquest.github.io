The Minecraft Server that runs on Bitcoin

Bitquest is a "Survival" style Minecraft server with a Bitcoin-denominated economy. Players can buy, sell and trade items using BTC micropayments with a fully funcional Bitcoin wallet you can send money to/from.

To start playing, Connect to the following address using the Desktop Version of Minecraft:

play.bitquest.co
